    Utilização do GIT Bash

  1. GIT ADD "ITEM"
    2. (Adição de html ou repositório para ser monitorado pelo git)
  2. GIT STATUS
    3. (Exibe as condições do diretório de trabalho e da área de staging)
  3. GIT LOG
    4. (Informará hash do commit)
  4. GIT CONFIG --LOCAL
    5. (Alteração de configuração para cada projeto)
  5. GIT CONFIG --GLOBAL
    6. (Alteração de configuração para a máquina toda)
  6. GIT LOG --ONELINE
    7. (Visualização de todos os commits em uma única linha)
  7. GIT LOG -P
    8. (Menos informação ainda de log)
  8. GIT LOG --PRETTY="FORMAT:%H"
    9. (traz o hash resumido seguido pela mensagem do commit)
  9. GIT INIT
    10. (Inicio de registro de repositório)
  10. GIT INIT --BARE
    11. (Indica que é repositório puro, contém apenas as alterações dos arquivos, e não uma cópia física de cada um dos arquivos).
  11. GIT REMOTE
    12. (Lista todos os repositórios locais)
  12. GIT REMOTE ADD LOCAL "CAMINHO"
    13. (Adiciona o seu repositório para a listagem, com nome de local)
  13. GIT PUSH LOCAL MASTER
    14. (Envia todas as alterações que fizemos, para o repositório "LOCAL")
  14. GIT PULL LOCAL MASTER
    15. (Puxa todas as alterações que estão no repositório "LOCAL")
  15. GIT COMMIT -M "TEXTO"
    16. (Realiza o Commit de todas as alterações realizadas)
  16. GIT BRANCH
    17. (Visualiza todos as "áreas de trabalho" do projeto)
  17. GIT BRANCH TITULO
    18. (Cria uma área de trabalho chamada "TITULO")
  18. GIT CHECKOUT TITULO
    19. (Altera sua área de trabalho Master, para Titulo)
  19. GIT CHECKOUT -B TITULO
    20. (Cria e Altera sua área de trabalho Master, cria a Titulo e muda para ela)
  20. GIT MERGE TITULO
    21. (Realiza o Commit, juntando a área de Trabalho, Master com Titulo)
  21. GIT REBASE TITULO
    22. (Pegará os commits na Branch Titulo, atualizando Master, possuindo todos os commits contidos em Titulo)

Obs:

GIT INIT SEMPRE UTILIZADO DENTRO DA PASTA COM O ITEM QUE VOCÊ DESEJA INICIAR O MONITORAMENTO

Exemplos:

GIT ADD INDEX.HTML

GIT CONFIG --LOCAL USER.NAME "BRIAN EVANOVICK"

GIT CONFIG --LOCAL USER.EMAIL "BRIAN-XAVIER@HOTMAIL.COM"

GIT REMOTE ADD LOCAL C:/USERS/BRIAN.SANTOS/DOWNLOADS/DOCUMENTOS_BRIAN/ALURA/SERVIDOR/
